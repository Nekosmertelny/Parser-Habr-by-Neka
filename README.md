# Habr Top Daily Parser

Простой парсер топа статей за сутки с Habr.com без использования официального API.  
Собирает заголовки, просмотры/охват и полный текст статей изнутри.

**Важно:** Habr часто меняет верстку (классы, структуру HTML).  
Если парсер перестанет работать — скорее всего нужно обновить селекторы (классы).  
Это обычно занимает 2–5 минут: открыть статью → F12 → посмотреть актуальный класс контейнера текста.

## Что делает скрипт
- Загружает страницу топа за сутки: https://habr.com/ru/articles/top/daily/
- Находит заголовки и ссылки на статьи
- Переходит по каждой ссылке (первые 3 для примера)
- Вытаскивает полный текст статьи (все параграфы из основного блока)
https://workupload.com/file/JH8qqnmtLHZ
## Требования
- Python 3.9+
- Библиотеки:
  - requests
  - beautifulsoup4
  - lxml (рекомендуется для скорости)
